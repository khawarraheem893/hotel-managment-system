🎯 Key Features

1. Dynamic Pricing

Prices automatically increase based on:

  • Weekends: Saturdays & Sundays get 40-60% markup
  • Seasons: Holiday periods can get 100%+ markup

Example: Room costs $100/night normally → $150 on weekends (1.5× multiplier)

2. User Panel

  • View available rooms with real-time pricing
  • Book rooms for specific dates
  • View booking history
  • Cancel bookings

3. Admin Panel

  • Add, edit, and delete rooms
  • Set base prices
  • Configure weekend pricing (applies to Sat/Sun)
  • Configure seasonal pricing (custom date ranges)
  • View all bookings
  • Change booking status
  • View dashboard statistics

4. Security

  • Passwords hashed with bcryptjs
  • JWT authentication tokens
  • Role-based access control (user vs admin)
  • Protected API endpoints

🔌 API Endpoints (Backend)

All API calls go to http://localhost:5000/api/

Authentication

  • POST /auth/register - Create new user
  • POST /auth/login - Login user
  • GET /auth/profile - Get user profile (requires token)

Rooms

  • GET /rooms - Get all rooms with pricing
  • GET /rooms/:id - Get specific room details
  • GET /rooms/:id/availability - Check availability for dates

Bookings (Requires Login)

  • POST /bookings - Create new booking
  • GET /bookings - Get my bookings
  • GET /bookings/:id - Get booking details
  • PUT /bookings/:id - Cancel booking

Admin (Admin Only)

  • POST /admin/rooms - Add room
  • PUT /admin/rooms/:id - Edit room
  • DELETE /admin/rooms/:id - Delete room
  • POST /admin/pricing - Add pricing rule
  • GET /admin/bookings - View all bookings
  • PUT /admin/bookings/:id - Update booking status
  • GET /admin/stats - Get dashboard statistics
